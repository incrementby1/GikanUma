## Prerequisites

![Node](https://img.shields.io/badge/node-18+-green) ![NPM](https://img.shields.io/badge/npm-9+-blue)

- Node.js
- npm

## Installation

```bash
cd client # frontend
npm i
cd ../api # backend (if ever go with sqlite)
npm i
```

## Usage

### Frontend (React)

Runs the React Development Server.

```bash
cd client
npm run dev
```

### Backend (Express)

Runs the api server using nodemon.

```bash
cd api
npm run dev
```
